### Branchements
Les deux capteurs laser doivent etre branchés a l'Arduino de la facon suivante:
 - VIN -> 5V
 - GND -> GND
 - SCL -> SCL
 - SDA -> SDA
 - X -> SHT_LOX1 (par défaut, 7) pour le détecteur de gauche
 - X -> SHT_LOX2 (par défaut, 6) pour le détecteur de droite
